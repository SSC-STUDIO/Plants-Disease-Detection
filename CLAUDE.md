# Plants Disease Detection

## Project Overview
植物病害检测系统，基于深度学习技术实现高效的病害分类。

## Tech Stack
- Language: Python
- Dependencies: requirements.txt, pyproject.toml
- Entry point: main.py
- Config: config.py

## Development Rules
- Branch convention: work on `codex/ai-Plants-Disease-Detection` branch
- Always update CHANGELOG.md on releases
- Validate model changes with test dataset before committing
- Keep checkpoints/ organized by model version

## Code Style
- Follow PEP 8
- Type hints for function signatures
- Document model architecture changes in docs/

## Key Paths
- `models/` — model definitions
- `dataset/` — data loading and preprocessing
- `libs/` — utility libraries
- `checkpoints/` — trained model weights
- `config.py` — training/inference configuration
- `main.py` — entry point
- `data/` — raw data
