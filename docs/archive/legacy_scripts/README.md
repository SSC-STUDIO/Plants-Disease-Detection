# Legacy Scripts Archive

This folder documents historical, machine-specific scripts that were removed from the public workflow.

Those scripts contained old local paths or private-network host examples, so they are not kept in the repository. Prefer the portable commands in:

- `README.md`
- `docs/QUICKSTART.md`
- `docs/MODERN_BACKBONE_TRAINING.md`
- `DATASET_CARD.md`

New automation should accept paths, hostnames, and credentials through command-line arguments or environment variables.
