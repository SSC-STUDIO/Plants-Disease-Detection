# Code of Conduct

## Our Pledge

We want this project to be useful for students, researchers, and developers working on plant disease recognition. Contributors are expected to keep discussions respectful, constructive, and focused on improving the project.

## Expected Behavior

- Use welcoming and inclusive language.
- Be respectful of different experience levels and backgrounds.
- Give actionable technical feedback.
- Credit datasets, papers, and prior work clearly.
- Avoid overstating model capability for real agricultural diagnosis.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Sharing private data, credentials, or unclear-license datasets.
- Presenting educational model outputs as certified plant pathology advice.
- Repeated off-topic promotion or spam.

## Enforcement

Maintainers may edit, hide, or remove comments and may block contributors who repeatedly violate this code. For sensitive reports, contact the maintainers through the repository owner's GitHub profile.
